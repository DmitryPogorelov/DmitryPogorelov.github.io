Политика конфиденциальности для приложения "Записано - Сделано"

1. Данная политика объясняет назначение разрешений на доступ к ресурсам оборудования, которые запрашивает приложение "Записано - Сделано" (далее - Приложение).

2. Доступ к оборудованию "Камера".

Доступ к камере необходим для того, чтобы была возможность прямо в приложении добавлять в заметки фотографии, сделанные с помощью камеры. Фотоснимки, полученные приложением с камеры смартфона/планшета, используются только и исключительно для добавления этих снимков в заметки, созданные в Приложении. Приложение не передаёт снимки в другие приложения. Приложение сохраняет снимки в файловой системе смартфона/планшета в собственной папке, недоступной для других приложений.

3. Доступ к оборудованию "Память".

Доступ "Память" необходим Приложению для того, чтобы получить возможность читать графические файлы (картинки) из галереи устройства. Полученные графические файлы используются только и исключительно для добавления этих файлов (картинок) в заметки, созданные в Приложении. Приложение не передаёт графические файлы в другие приложения. Приложение сохраняет графические файлы в файловой системе смартфона/планшета в собственной папке, недоступной для других приложений.

4. Удаление данных

При удалении заметки удаляется ВСЯ связанная с заметкой информация, включая графические файлы. После удаления заметки восстановить ее невозможно.

5. Хранение данных

Все данные пользователя, включая графические файлы, хранятся локально на смартфоне/планшете и НЕ передаются в другие хранилища. Повреждение или утрата устройства приведуту к утрате ВСЕХ данных приложения.
